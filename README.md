#  Authy

##  Setup



### Running the Application

### Server
 navigate to the server directory
```sh
cd nodejs-mongo-sendgrid
```
To run the server:

1. Install dependencies:
   ```
   yarn install
   ```
   

```sh
yarn run nodemon
```


### Frontend
    
 navigate to the frontend directory
 ```sh
    cd next-authentication
```

To run the frontend:

1. Install dependencies:
   ```
   yarn install
   ```

```
yarn run dev
```
